Somatório

Algoritmo simples para soma de valores.

Em JavaScript, o método map é usado para criar um novo array com base em um array original, aplicando uma função a cada elemento da matriz original.

Quando usamos o método map em um array de valores numéricos, podemos passar uma função como parâmetro que converte cada elemento para o tipo Number. Isso é útil quando a matriz original contém valores que não são do tipo numérico, como strings.

isNaN() é uma função JavaScript que verifica se um valor é um número. O nome isNaN é uma abreviação para "is Not a (Valid) Number", ou seja, "não é um número (válido)" em português. Ela retorna true se o valor não for um número e false se o valor for um número ou puder ser convertido em um.
